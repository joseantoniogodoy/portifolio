# Portfólio acessível — José Antonio Godoy

## Sobre
Este projeto é a refatoração de um site de portfólio, com foco em **acessibilidade** e **melhoria da experiência do usuário**, usando HTML, CSS e JavaScript.

O site apresenta formações e registros fotográficos (cards, modais e carrosséis), além de um menu de acessibilidade para facilitar a navegação.

## Recursos de acessibilidade implementados
- **Menu de acessibilidade** (abre/fecha opções)
- **Ajuste de tamanho de fonte** (A+ / A-) com persistência
- **Alto contraste** (modo alternável)
- **Atributos ARIA** (aria-label, aria-expanded, aria-controls, aria-pressed)
- **Navegação por teclado** (tabindex em seções e cards, foco visível)
- **Skip link** (“Pular para o conteúdo principal”)
- **Textos alternativos (alt)** nas imagens
- **Foco retorna ao gatilho** após fechar modal

## Animações e desempenho
- Animações de entrada com **ScrollReveal** (via CDN) em seções principais do site, respeitando a preferência do usuário por **redução de movimento**.

## Tecnologias utilizadas
- Bootstrap (CSS/JS)
- Bootstrap Icons
- ScrollReveal.js
- HTML
- CSS
- JavaScript

## Como executar
Basta abrir o arquivo `Portifolio/index.html` no navegador.
